# About

**Astro Baseline is a minimal Astro baseline focused on structure, clarity, and build-time decisions.**

This starter intentionally avoids UI kits, heavy abstractions, and runtime JavaScript. It provides a solid foundation for websites that value performance, accessibility, and long-term maintainability.

You are expected to adapt it, extend it, and make it your own.
